## Purpose
Wanted to get some initial experiance with React and CSS Grid

## Usage
1. Clone repo
2. Run `npm start`
3. Select board size
4. Manually put in [Oh h1](http://0hh1.com/) puzzle
5. Press solve button
6. Watch program solve puzzle
7. Congratulations! You have cheated succesfully!
